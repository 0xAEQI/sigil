# Notes — The Deeper Vision

Raw capture, to be folded into ARCHITECTURE-V4 later.

## The Insight

Notes are not tasks. Notes are not chat messages. Notes are **directives that reshape what Sigil is.**

Chat = interactive. Linear. You act, it responds. Ephemeral.
Notes = imprinting. You write your will. Sigil **consumes** it. The note becomes Sigil's DNA.

## How It Works

A note doesn't trigger a task and disappear. It persists as source of truth.
Sigil continuously references it. The total of all your notes = what Sigil has become.

```
Note: "I want a profitable trading operation that runs 24/7 with <1% drawdown"

This doesn't create one task. It reshapes Sigil:
  - Trader agent identity updated to prioritize risk management
  - New watchdog: drawdown monitor with alert threshold
  - New mission: optimize trading parameters for consistency
  - Memory updated: this is a core goal, not a one-off request
  - Future tasks in this project are evaluated against this directive
```

## Notes vs Chat

| Chat | Notes |
|------|-------|
| "Fix the trading bot" → one task, done | "Profitable 24/7 trading" → ongoing directive |
| Linear conversation | Non-linear thought dump |
| Ephemeral (scrolls away) | Persistent (the constitution) |
| Sigil responds | Sigil evolves |
| Interactive | Imprinting |

## The Note Lifecycle

1. You write raw thoughts (unstructured, messy, brain dump)
2. Sigil consumes the note — parses intent, extracts directives
3. Sigil evolves: creates agents, projects, missions, memory, watchdogs, identity changes
4. Sigil proactively comes back: "I read your note about X. Should I do Y?"
5. The note stays. Sigil keeps referencing it for future decisions.
6. You update the note. Sigil re-consumes, adapts.

## The Key Realization

Your notes ARE sigils. Literally. Symbols of intent that manifest into reality.
The product is an Obsidian-like note tool where your notes actually do things.

This complements (doesn't replace) the chat interface:
- Chat: interactive, immediate, conversational
- Notes: reflective, persistent, constitutional

You want to produce as many thoughts as possible at any given moment without
cluttering the linear flow of conversation. That's what notes are for. Then Sigil
processes them asynchronously, on its own time, and comes back to you.
